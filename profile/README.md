# 반갑습니다! 👋
SPI 개발팀에 합류하시면 다양한 경험을 가진 동료들 사이에서 서로의 경험과 기술을 공유하며 빠르게 성장 가능하며, 사용자 및 동료들에게 필요한 것에 대해서 고민하고 기술로 구현하는 과정을 겪으실 수 있습니다.

만약 SPI 개발팀에 합류하시면 지금까지 없었던 새로운 서비스를 만들어가면서 업무에 깊게 몰입할 수 있을 것입니다.
